# ITO Booking Form

Booking form as WordPress widget for fastboat, tour, and flight from [Indo Travel Online](http://www.indotravelonline.com).

- [x] Fastboat
- [x] Tour
- [x] Flight

## Shortcodes
**[itobf_fb]** for Fastboat  
**[itobf_tr]** for Tour  
**[itobf_fl]** for Flight
